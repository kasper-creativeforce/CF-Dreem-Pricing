# Dreem Design Tokens

## Colors
- Accent Green: `#3ecf6e` (UI) / `#37CD7D` (logo mark)
- Page Background: `#060606`
- Surface: `#121212`
- Elevated: `#1a1a1a`
- Border: `#2b2b2b`
- Subtle: `#414141`
- Text Primary: `#fcfcfd`
- Text Muted: `rgba(252,252,253,0.50)`

## Typography
- Headings/UI: Space Grotesk, sans-serif
- Body/captions: Inter, sans-serif
- Never swap these roles

## Components
- Card radius: 15px
- Button radius: 7px
- Card border: `1px solid #2b2b2b`
- Primary button: background `#3ecf6e`, text `#060606`
- Ghost button: border `#414141`, text `#fcfcfd`

## Check icon color (in dark Dreem rows)
- `#3ecf6e`
